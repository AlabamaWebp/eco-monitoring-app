# Demo Script (5–7 minutes)

## 1) Dashboard (30–45 сек)

- Открыть `Dashboard`.
- Показать карточки: полигоны, датчики, измерения, CSV-загрузки.
- Показать блок последних импортов.

## 2) Справочники (30–45 сек)

- Открыть страницу `Справочники`.
- Коротко показать:
  - полигоны;
  - типы датчиков;
  - единицы измерения.
- Подчеркнуть, что это нормализованные справочники.

## 3) Показ CSV (30 сек)

- Открыть `sample_data/polygon_1_measurements.csv`.
- Показать колонку `Дата` и несколько датчиков.

## 4) Импорт CSV (1 мин)

- Перейти на страницу `Загрузка CSV`.
- Выбрать полигон, указать фамилию, выбрать CSV.
- Нажать `Загрузить CSV`.
- Показать результат: `rows_count`, `measurements_count`, `skipped_values`.

## 5) Таблица измерений (1 мин)

- Перейти на страницу `Измерения`.
- Показать данные после импорта.
- Применить фильтр по полигону и датчику.
- Переключить сортировку по дате.

## 6) Графики (1.5–2 мин)

- Открыть страницу `Графики`.
- Режим 1: выбрать полигон + несколько датчиков (например, CO2 и влажность).
- Нажать `Период sample-данных`, затем построить график.
- Режим 2: выбрать один датчик + несколько полигонов.
- Показать легенду, tooltip, dataZoom.

## 7) Коротко про БД (45 сек)

- Пояснить, что `measurements` — главная таблица.
- `import_files` хранит только историю загрузок.
- Архитектура в 3НФ за счёт выноса справочников в отдельные таблицы.
